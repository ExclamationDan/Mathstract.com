Vector Applications

Force and direction

By definition a vector is a quantity having direction as well as magnitude, especially as determining the position of one point in space relative to another.

For the following application, a vector is used to represent an object's velocity. That is, the speed and direction of an object.Vectors are usually denoted as a boldened U or V, with an optional arrow 'hat' overtop itself.

Vector definition and assignment notation follows as vector: V = <x,y,z,...>
This is similar to how a point 'P' is defined where P = (x,y,z,...) Vector additon shares all properties of basic addition.

Vector Additon

Consider vectors V and U where:
V = <1,2> and U = <3,4>

V + U = <1+3, 2+3> = <4,5>

Note that vector addition is also commutative,(V + U) = (U + V)
